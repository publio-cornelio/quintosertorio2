Soporte técnico de verba volant

Este proyecto consta de tres componentes diferenciados:

  • Una pagina de presentacion realizada mediante html, css y un poco de javascript.
  • Este sitio web creado con un generador de paginas a partir de markdown, y con un estilo propio de los gereradores de documentacion en linea.
  • Un blog creado por el mismo generador de paginas web, pero con un estilo mas acorde con una comunicacion mas informal.

El objetivo de esta web es mostrar todos los aspectos tecnicos que hay detras de este proyecto, para facilitar su mantenimiento y replica.
